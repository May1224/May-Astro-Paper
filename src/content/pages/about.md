AstroMay 是一个用于整理前端学习、工程实践和个人成长记录的中文技术博客。

## 关于本站

网站基于 AstroPaper 和 Astro 6 构建，内容使用 Markdown 管理，源码保存在 GitHub，并由 GitHub Actions 自动发布到 GitHub Pages。

当前内容主要包括：

- **前端学习**：JavaScript、Vue、浏览器、网络和工程化笔记。
- **实践记录**：开发、部署和工具使用过程中遇到的问题与解决过程。
- **项目复盘**：对真实完成的功能进行背景、实现和改进总结。
- **阶段记录**：记录博客建设和学习计划的变化。

## 内容原则

- 先说明问题和结论，再展开实现细节。
- 代码示例尽量可以独立验证，并标明必要的运行条件。
- 图片用于解释结构、流程和结果，不使用无关素材填充页面。
- 对持续更新的文章标明更新时间，旧结论发生变化时直接修订。
- 不公开账号、密钥、私人对话和其他敏感信息。

## 技术与发布

站点保持为纯静态架构，不引入数据库、评论系统、访问统计和内容管理后台。每次推送到 GitHub 后会自动完成类型检查、静态构建、Pagefind 搜索索引和 Pages 发布。

这种结构的重点是可维护：文章和站点代码使用同一套版本记录，构建结果可以提前验证，也不需要长期维护服务器。

## 后续方向

第一阶段继续完善文章质量和阅读体验。等真实项目资料准备完成后，再增加项目展示页，并补充更完整的个人介绍和公开联系方式。
